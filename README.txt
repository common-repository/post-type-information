Display information about your custom post types on your dashboard.

== Description ==

Add information about all your custom post types and taxonomies in the At a Glance section of the Dashboard.

= Features =

* Allows you to select which post types and taxonomies to add via settings page.
* Shows a count of all published posts per type in the dashboard, and links to the post type edit page.
* Shows a count of all terms per taxonomy and links to the taxonomy page.

== Installation ==

1. Upload `gs-post-type-information.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Settings >> Post Type Information to select the additional post types and taxonomies to display.
